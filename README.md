# Netflix Clone Website

A responsive Netflix clone built using **HTML and CSS with media queries**.  
It replicates the look and feel of Netflix’s homepage, including banners, navigation, and responsive layouts for different screen sizes.

## 🚀 Features
- Responsive design using CSS media queries
- Netflix‑style navigation bar
- Hero banner section
- Movie/TV show grid layout
- Footer with links
- Pure HTML and CSS (no JavaScript)
